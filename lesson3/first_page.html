Массивы

Массив - это тип данных, содержащий пронумерованные значения. Каждое пронумерованное значение значение называется элементом массива, а число с котторым связан элемент индексом. Так как JavaScript нетипизированный язык, элемент массива может иметь любой тип, причем разные элементы одного массива могут иметь разные типы. Элементы массива могут даже содержать другие массивы, что позволяет создавать массивы массивов. Пример создания массивов: 

        var empty = []                  // Пустой массив
        var primes = [1,56,87,909,61,6] // Массив с пятью числовыми элементами
        var misc = [1.1, true, 'a']     // Три элемента разных типов

JavaScript поддерживает две разновидности массивов:

  • Ассоциативный массив (Hash), где данные хранятся по произвольному ключу.
  • Числовой массив Array, где данные хранятся по номерам.

Также можно создать массив с помощью конструктора Array( ), вызвать его можно тремя способами:

  • Вызов конструктора без аргументов:
    var a = new Array()

    Создан пустой массив, эквивалентный литералу [ ].

  • Конструктору явно указываются значения первых n элементов массива:
    var a = new Array(5, 4, 3, 2, 1, 'testing, testing')

    В этом случае конструктор получает список аргументов. Нумерация элементов начинается с 0. Свойство lenght массива устанавливается равным количеству элементов, переданных конструктору.

  • Вызов с единственным числовым аргументом, определяющим длинну массива:
    var a = new Array(10);

    Эта форма может быть использована для предварительного размещения массива если его длинна известна заранее. Все элементы в массиве имеют значение undefined.

Индекс массива – целое число, либо значение типа, приводимого к целому, указывающее на конкретный элемент массива. Доступ к элементам массива осуществляется с помощью оператора - [ ]. Слева от скобок должна находиться ссылка на массив. Внутри скобок должно находиться произвольное выражение, имеющее неотрицательное целое значение. Этот синтаксис пригоден как для чтения, так и для записи значения элемента массива. 

        var array = ["text", 234, "Alex", true, 435];  
        
        array[1]=234; // Обращаемся ко второму элементу

Добавление и удаление новых элементов в массиве

В Javascript количество элементов массива может изменяться в любой момент. Чтобы добавить новый элемент достаточно присвоить ему значение:

a[10] = 10;

Массивы в JavaScript могут быть разреженными, это значит что индексы массива не обязательно принадлежат непрерывному диапазону чисел. Памяты выделяется только для тех элементов которые хранятся фактически. В следующем примере JavaScript выделяет место в памяти только для определенных элементов 0 и 10.000 но не выделяет место для оставшихся 9.999:

a[0] = 1;
a[10000] = 'это элемент 10.000'

Оператор delete записывает в элемент массива undefined, при этом сам элемент массива продолжает свое существование. Для удаления элементов так, чтобы остающиеся элементы сместили к началу массива необходимо воспользоваться методом Array.shift( ), метод Array.pop( ) - удаляет последний элемент массива, а Array.splice( ) - непрерывный диапазон элементов.

Длинна массива

Все массивы, как созданные с помощью конструктора Array( ), так и определенные с помощью литерала массива, имеют специальное свойство lenght, устанавливающее количество элементов в массиве. Поскольку массивы могут иметь неопределенные элементы , более точная формулировка звучит так: свойство lenght всегда на еденицу больше чем самый большой номер элемента массива. Свойство lenght массива автоматически обновляется, оставаясь корректным при добавлении новых элементов в массив. 

        var a = new Array (); // a.lenght == 0 (ни один элемент не определен)
        a = new Array(10);     // a.lenght == 10 (определены пустые элементы 0-9)
        a = new Array(1,2,3);  // a.lenght == 3 (определены элементы 0-2)
        a = [4, 5];            // a.lenght == 2 (определены элементы 0 и 1)
        a[5] = -1;             // a.lenght == 6 (определены элементы 0, 1 и 5)
        a[49] = 0;             // a.lenght == 50 (определены элементы 0, 1, 5 и 49)

Методы массивов

Метод join( ) - преобразует все элементы массива в строки и объединяет их. Можно указать необязательный строковый аргумент, предназначенный для разделения элементов в результирующей строке. Если разделитель не задан, используется запятая.

var a = [1, 2, 3]; // Создаем новый массив с тремя элементами
var s = a.join(); // s == "1, 2, 3"

Метод reverse( ) - меняет порядок следования элементов в массиве на противоположный и возвращает массив с переставленными элементами. Пример:

var a = new Array(1, 2, 3) // a[0] = 1, a[1] = 2, a[2] = 3
a.reverse(); // теперь a[0] = 3, a[1] = 2, a[2] = 1
var s = a.join(); // s == "3, 2, 1"

Метод sort( ) - сортирует элементы массива. Если метод sort( ) вызывается без аргументов, то он сортирует элементы массива в алфавитном порядке (при необходимости временно преобразуя их в строки для выполнения сравнения):

var a = new Array('banana', 'cherrry', 'apple');
a.sort();
var s = a.join(', '); // s == 'apple, banana, cherry'

Для сортировки в каком-либо другом порядке можно передать методу sort( ) в качестве аргумента функцию сравнения. Эта функция устанавливает, какой из двух ее аргументов должен присутствовать раньше в отсортированном списке. Если первый аргумент должен предшествовать второму, функция сравнения возвращает отрицательное число. Если первый аргумент в отсортированном массиве должен следовать за вторым, то функция возвращает число больше нуля. А если два значения эквивалентны (т.е. порядок их расположения не важен), функция сравнения возвращает 0. Поэтому для сортировки элемента в числовом порядке, а не в алфавитном можно сделать следующее: 

        var a = [33, 4, 1111, 222]; 
        a.sort();                    // Алфавитный порядок: 1111, 222, 33, 4 
        a.sort(function(a, b) {      // Числовой порядок: 4, 33, 222, 1111 
                return a-b;          // Возвращает значение < 0, 0, или > 0 
            });                      // в зависимости от порядка сортировки a и b

Метод concat( ) - создает и возвращает новый массив, содержащий элементы исходного массива, для которого был вызван метод concat( ), последовательно дополненный значениями всех аргументов, переданных методу concat( ). Если какой-либо из этих аргументов сам является массивом, в результирующий массив добавляются новые элементы. 

        var a = [1, 2, 3];
        a.concat(4, 5)               // Возвращает [1, 2, 3, 4, 5]
        a.concat([4, 5])             // Возвращает [1, 2, 3, 4, 5]
        a concat([4, 5],[6, 7])      // Возвращает [1, 2, 3, 4, 5, 6, 7]
        a.concat(4, [5,[6,7]])       // Возвращает [1, 2, 3, 4, 5, [6, 7]]

Метод slice( ) - возвращает фрагмент, или подмассив, указанного массива. Два аргумента определяют начало и конец возвращаемого фрагмента. Возвращаемый массив содержит элемент, номер которого указан в качестве первого аргумента, плюс все последующие элементы, вплоть до элемента который указан во втором аргументе. Если указан только один аргумент, возвращаемый массив содержит все элементы от начальной позиции до конца массива. Если какой-либо из аргументов отрицателен, он задает номер элемента массива относительно конца массива. Так аргумент равный -1 задает последний элемент массива, а аргумент равный -3 третий элемент массива с конца. 

        var a = [1, 2, 3, 4, 5];
        a.slice(0, 3);               // Возвращает [1, 2, 3]
        a.slice(3);                  // Возвращает [4, 5]
        a.slice(1, -1);              // Возвращает [2, 3, 4]
        a.slice(-3, -2);             // Возвращает [3]

Метод splice( ) - это универсальный метод для вставки или удаления элементов массива, метод может удалять элементы из массива, вставлять новые элементы в массив или выполнять обе операции одновременно. Первый аргумент задает позицию в массиве, с которой начинается вставка и/или удаление. Второй аргумент задает количество элементов которые должны быть удалены из массива. Если второй элемент не указан то удаляются все элементы от указанного до конца массива. 

        var a = [1, 2, 3, 4, 5, 6, 7, 8]; 
        a.splice(4);                      // Возвращает [5, 6, 7, 8]; a равно [1, 2, 3, 4]
        a.splice(1, 2);                   // Возвращает [2, 3]; a равно [1, 4]
        a.splice(1, 1);                   // Возвращает [4]; a равно [1]

Первые два аргумента задают элементы массива подлежащие удалению. За этими аргументами может следовать любое количество дополнительных аргументтов которые будут указывать сколько элементов будет вставлено в массив. 

        var a = [1, 2, 3, 4, 5];   
        a.splice(2, 0, 'a', 'b');         // Возвращает [ ]; a равно [1, 2, 'a', 'b', 3, 4, 5]
        a.splice(2, 2, [1, 2], 3)         // Возвращает ['a', 'b']; a равно [1, 2,[1, 2], 3, 3, 4, 5]

Методы push( ) и pop( ) - позволяет работать с массивами как со стеками. Метод push( ) добавляет оин или несколько новых элементов в конец массива и возвращает его новую длинну. Метод pop( ) выполняет обратную операцию - удаляет последний элемент массива, уменьшает длинну массива, и возвращает удаленное им значение. 

        var stack = [ ];             // стек: [ ]
        stack.push(1, 2);            // стек: [1, 2]     Возвращает 2
        stack.pop();                 // стек: [1]        Возвращает 2
        stack.push(3);               // стек: [1, 3]     Возвращает 2
        stack.pop();                 // стек: [1]        Возвращает 3
        stack.push([4, 5]);          // стек: [1,[4, 5]] Возвращает 2
        stack.pop();                 // стек: [1]        Возвращает [4, 5]
        stack.pop();                 // стек: [ ]        Возвращает 1

Методы shift( ) и unshift( ) - работает так же как и предыдущий метод за исключением того, что они вставляют и удаляют элементы в начале массива, а не в его конце. 

        var a = [ ];                 // a:[ ]
        a.unshift(1);                // a:[1]           Возвращает: 1
        a.unshift(22);               // a:[22, 1]       Возвращает: 2
        a.shift();                   // a:[1]           Возвращает: 22
        a.unshift(3, [4, 5]);        // a:[3,[4, 5],1]  Возвращает: 3
        a.shift();                   // a:[[4, 5], 1]   Возвращает: 3
        a.shift();                   // a:[1]           Возвращает: [4, 5]
        a.shift();                   // a:[ ]           Возвращает: 1

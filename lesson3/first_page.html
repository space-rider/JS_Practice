Массивы

Массив - это тип данных, содержащий пронумерованные значения. Каждое пронумерованное значение значение называется элементом массива, а число с котторым связан элемент индексом. Так как JavaScript нетипизированный язык, элемент массива может иметь любой тип, причем разные элементы одного массива могут иметь разные типы. Элементы массива могут даже содержать другие массивы, что позволяет создавать массивы массивов. Пример создания массивов: 

        var empty = []                  // Пустой массив
        var primes = [1,56,87,909,61,6] // Массив с пятью числовыми элементами
        var misc = [1.1, true, 'a']     // Три элемента разных типов

JavaScript поддерживает две разновидности массивов:

  • Ассоциативный массив (Hash), где данные хранятся по произвольному ключу.
  • Числовой массив Array, где данные хранятся по номерам.

Также можно создать массив с помощью конструктора Array( ), вызвать его можно тремя способами:

  • Вызов конструктора без аргументов:
    var a = new Array()

    Создан пустой массив, эквивалентный литералу [ ].

  • Конструктору явно указываются значения первых n элементов массива:
    var a = new Array(5, 4, 3, 2, 1, 'testing, testing')

    В этом случае конструктор получает список аргументов. Нумерация элементов начинается с 0. Свойство lenght массива устанавливается равным количеству элементов, переданных конструктору.

  • Вызов с единственным числовым аргументом, определяющим длинну массива:
    var a = new Array(10);

    Эта форма может быть использована для предварительного размещения массива если его длинна известна заранее. Все элементы в массиве имеют значение undefined.

Индекс массива – целое число, либо значение типа, приводимого к целому, указывающее на конкретный элемент массива. Доступ к элементам массива осуществляется с помощью оператора - [ ]. Слева от скобок должна находиться ссылка на массив. Внутри скобок должно находиться произвольное выражение, имеющее неотрицательное целое значение. Этот синтаксис пригоден как для чтения, так и для записи значения элемента массива. 

        var array = ["text", 234, "Alex", true, 435];  
        
        array[1]=234; // Обращаемся ко второму элементу

Добавление и удаление новых элементов в массиве

В Javascript количество элементов массива могет изменяться в любой момент. Чтобы добавить новый элемент достаточно присвоить ему значение:
a[10] = 10;
